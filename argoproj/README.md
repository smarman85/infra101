# Arog Ecosystem

## Tools used:
- ArgoCd
- Argo Workflows
- Argo Events
- Kind

## Usage:
This implements a basic kind cluster. Running `make init` will build a new kind cluster and install a base implementation of Argo Events, workflows, and CD.
